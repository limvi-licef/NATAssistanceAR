async Task Cameras.FrameReaderCamera.Init ( MediaStreamType  streamType,
MediaFrameSourceKind  sourceKind 
)
inline

Initialize the camera. Check available cameras before build the FrameReader object and define the output format. Main camera could be initialized using MediaStreamType.VideoRecord and MediaFrameSourceKind.Color as arguments.

Parameters
streamTypeType of media that the media reader will be used for. Values : VideoRecord (recommanded for accessing frame), VideoPreview, Audio, Photo.
sourceKindFor cameras, specify the type of image produced. Values : Color (recommanded), Infrared, Depth.
